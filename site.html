Raisin Dettre

This place is not a place of honor... no highly esteemed deed is commemorated here... nothing valued is here.

What is here is dangerous and repulsive to us. This message is a warning about danger.

Sandia Report SANDIA92-1382 "Expert Judgement on Markers to Deter Inadvertent Human Intrusion into the Waste Isolation Pilot Plant", Trauth, Hora and Guzowski, 1993, Page F-49.

The web died several years ago, but like any giant dinosaur it takes time for the body to react to the death of the brain. I wrote my first web page back in 1995 or so, and I've treated it as a public place to occasionally put my thought, mostly lots of photos and essays for when I lived in America and I wanted to share my stories with relatives and friends back home, before Facebook or Instagram existed.

It is usually where I host my teaching materials for courses at Leiden Observatory, digital copies of all my publications that are otherwise stuck behind paywalls, and the public face for Workshops, Meetings and my research ideas.

Starting with writing HTML from scratch, I've gone through my own custom scripts in Perl, a blog system based on the bash shell, then settled on dokuwiki to serve Markdown pages in a pleasing format for almost a decade. I have been using org mode in Emacs for all my calendar appointments and scheduling, and since Emacs has a large hacker community supporting its development, there is inevitably an option to write org pages and have them be exported to HTML. I've been a long time enthusiast of Edward Tufte's books on style and graphics, liking the relative simplicity and clean style, so when I discovered Tufte CSSPretentious, moi?, I've been trying to get emacs, org-mode, ox-tufte and tufte-css to play nicely together.

I have failed miserably.

Wondring how to solve this problem, I had a sudden burst of clarity and insight - I'm messing around with an unstable compilation chain using three different code bases, when I can quite happily bash out HTML by hand. So that's what I'm going back to. This is all written direct into HTMLWell, I'm using the Atom editor which makes closing the tags and checking this into my GitHub account very easy., and synced up to the web server space at work. I'll get back to coding for astronomy projects and papers, and more time for making bread.
